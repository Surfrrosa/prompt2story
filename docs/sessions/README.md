# Session Logs

Session logs track what was accomplished, decisions made, and next steps for each development session.

## Format

Each session log should include:
1. Date and status
2. What was accomplished
3. Next tasks (prioritized)
4. Any blockers
5. Key decisions made
6. Known issues discovered
